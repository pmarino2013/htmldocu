🚀 Clase 3: Recursos Multimedia en HTML

Tema: Imágenes, enlaces, listas y rutas (relativas vs absolutas)

🎯 Objetivo de la clase

  • Al terminar esta clase el estudiante podrá:
  • Insertar imágenes en una página web.
  • Crear enlaces que lleven a otras páginas.
  • Organizar información usando listas.
  • Entender cómo funcionan las rutas (paths).
  • Evitar errores comunes cuando algo “no se ve”.

🧠 1. ¿Qué es un recurso multimedia?

En la web, no todo es texto.

Un recurso multimedia puede ser:

  • 🖼️ Imágenes
  • 🎥 Videos
  • 🎵 Audio
  • 🔗 Enlaces
  • 📄 Archivos

Hoy trabajaremos con:

  • Imágenes
  • Enlaces
  • Listas

🖼️ 2. Imágenes en HTML

En HTML, las imágenes se agregan con la etiqueta:

<img>

Pero esta etiqueta necesita información adicional.

🔹 Sintaxis básica

<img src="imagen.jpg" alt="Descripción de la imagen">

📌 ¿Qué significa cada parte?

Atributo Qué hace
src Indica dónde está la imagen
alt Texto alternativo si la imagen no carga

🔎 Ejemplo real

<img src="perfil.jpg" alt="Foto de perfil de Juan Pérez">

🎯 ¿Por qué es IMPORTANTE el alt?

Porque:

  • Mejora la accesibilidad (lectores de pantalla)
  • Ayuda al SEO
  • Se muestra si la imagen falla

💡 Explicación simple:
El alt es como describirle la imagen a alguien que no puede verla.

📏 Buenas prácticas

❌ Evitar:

<img src="foto.png">

✅ Mejor:

<img src="foto.png" alt="Foto profesional con fondo blanco">

🔗 3. Enlaces (Links)

Los enlaces son lo que hace que la web sea una red.

Se crean con la etiqueta:

<a>

🔹 Sintaxis básica

<a href="https://google.com">Ir a Google</a>

📌 Partes importantes

Elemento Función
href Dirección a la que vamos
Texto interno Lo que el usuario ve

🌍 Abrir en nueva pestaña

<a href="https://google.com" target="_blank">
  Ir a Google
</a>

target="_blank" → abre en otra pestaña.

📧 Enlace para enviar email

<a href="mailto:correo@gmail.com">Enviar correo</a>

📞 Enlace para llamar

<a href="tel:+5493811234567">Llamar ahora</a>

Perfecto para páginas móviles 📱

📋 4. Listas en HTML

Las listas organizan información.

Hay 3 tipos:

🔹 1️⃣ Lista no-ordenada (ul)

Para elementos sin orden específico.

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Resultado:
• HTML
• CSS
• JavaScript

🔹 2️⃣ Lista ordenada (ol)

Cuando importa el orden.

<ol>
  <li>Encender la computadora</li>
  <li>Abrir VS Code</li>
  <li>Escribir código</li>
</ol>

Resultado:

  1. Encender la computadora
  2. Abrir VS Code
  3. Escribir código

🔹 3️⃣ Lista descriptiva (dl)

Para definiciones.

<dl>
  <dt>HTML</dt>
  <dd>Lenguaje de estructura web</dd>

  <dt>CSS</dt>
  <dd>Lenguaje de estilos</dd>
</dl>

🗂️ 5. Rutas: Relativas vs Absolutas

Esta parte suele confundir. Vamos a explicarlo simple.

🏠 Pensalo así:

Tu proyecto es como una casa.

Las rutas indican cómo llegar a una habitación.

🌍 Ruta Absoluta

Es la dirección completa.

Ejemplo:

<img src="https://misitio.com/imagenes/foto.jpg">

Es como decir:

"Vivís en Argentina, Tucumán, Monteros, Calle X 123"

Siempre funciona, aunque cambies de lugar.

📁 Ruta Relativa

Es la dirección dentro de tu proyecto.

Si tu estructura es:

proyecto/
 ├── index.html
 ├── imagenes/
      └── perfil.jpg

Entonces:

<img src="imagenes/perfil.jpg">

Eso significa:

"Entrá a la carpeta imagenes y buscá perfil.jpg"

🔼 Subir un nivel

Si estás en una carpeta y querés subir:

../

Ejemplo:

<img src="../imagenes/foto.jpg">

🎯 Regla de oro

Si la imagen no aparece:

  • Revisar nombre exacto
  • Revisar mayúsculas/minúsculas
  • Revisar ruta

El 90% de los errores están ahí.

🧩 6. Ejemplo Integrador

Vamos a crear una mini biografía profesional.

<!DOCTYPE html>
<html>
<head>
  <title>Mi Biografía</title>
</head>
<body>

  <h1>Juan Pérez</h1>

  <img src="imagenes/perfil.jpg" alt="Foto de Juan Pérez">

  <h2>Sobre mí</h2>
  <p>Soy desarrollador web en formación.</p>

  <h2>Habilidades</h2>
  <ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>Trabajo en equipo</li>
  </ul>

  <h2>Contacto</h2>
  <a href="mailto:juan@email.com">Enviar Email</a>

</body>
</html>

⚠️ Errores Comunes de Principiantes

❌ No cerrar etiquetas

❌ Escribir mal el nombre de la imagen

❌ Olvidar el atributo alt

❌ Poner espacios en nombres de archivos (ej: "foto perfil.jpg")

Mejor usar:

foto-perfil.jpg

🎥 Incorporar Videos de YouTube en HTML

Una página web no solo puede tener imágenes y texto… también puede tener videos.

Y el método más común es insertar videos desde YouTube usando un <iframe>.

🧠 ¿Qué es un <iframe>?

Un <iframe> es una etiqueta que permite mostrar contenido externo dentro de tu página web.

Es como “abrir una ventanita” hacia otro sitio.

🔹 Cómo insertar un video de YouTube paso a paso

1️⃣ Ir al video en YouTube

2️⃣ Hacer clic en “Compartir”

3️⃣ Elegir “Insertar”

4️⃣ Copiar el código que aparece

YouTube te dará algo como esto:

<iframe width="560" height="315" 
src="https://www.youtube.com/embed/ID_DEL_VIDEO" 
title="YouTube video player" 
frameborder="0" 
allowfullscreen>
</iframe>

📌 Explicación simple de cada parte

Atributo Qué hace
width Ancho del video
height Alto del video
src Dirección del video (muy importante)
title Describe el contenido (accesibilidad)
allowfullscreen Permite pantalla completa

⚠️ IMPORTANTE

No se usa la URL normal del navegador:

❌ Incorrecto:

https://www.youtube.com/watch?v=abc123

✅ Correcto:

https://www.youtube.com/embed/abc123

YouTube transforma automáticamente la URL cuando copiás desde “Insertar”.

🧩 Ejemplo práctico en una biografía profesional

<section>
  <h2>Mi presentación en video</h2>

  <iframe 
    width="560" 
    height="315" 
    src="https://www.youtube.com/embed/abc123"
    title="Video de presentación"
    allowfullscreen>
  </iframe>

</section>

📱 ¿Y en dispositivos móviles?

Más adelante, cuando vean Responsive Design, aprenderán a hacer que el video:

  • Se adapte al ancho de pantalla
  • No se desborde
  • Mantenga proporción

Pero por ahora, alcanza con entender cómo insertarlo.

🚀 Buenas prácticas

✔ Siempre usar el código que provee YouTube
✔ Agregar title descriptivo
✔ Colocar el video dentro de una sección con encabezado
✔ No abusar de demasiados videos en una misma página

🧠 Concepto importante

Cuando usamos <iframe>, estamos mostrando contenido externo, no lo estamos guardando en nuestro servidor.

Eso significa:

  • El video sigue alojado en YouTube
  • Si YouTube elimina el video, desaparece de nuestra web
  • No estamos descargando el archivo
