# DocuHTML: HTML para principiantes

## Descripción

Este proyecto es un curso introductorio a HTML (HyperText Markup Language) diseñado para principiantes. El HTML es el lenguaje que organiza títulos, párrafos, imágenes y enlaces para que una página cobre vida en la web. No es solo "etiquetas raras": es el mapa que guía a tu navegador para construir cada sitio que visitas.

El curso está estructurado en clases que te enseñan a leer la web como un desarrollador, con ejemplos de etiquetas, estructura y mucha curiosidad.

## Estructura del proyecto

- `index.html`: Página principal del curso con introducción y navegación.
- `pages/clase2.html`: Segunda clase del curso.
- `pages/clase3.html`: Tercera clase del curso.
- `pages/clase4.html`: Cuarta clase del curso.

## Cómo usar

1. Clona o descarga este repositorio.
2. Abre `index.html` en tu navegador web preferido.
3. Navega a través de las clases usando los enlaces proporcionados.

No se requieren dependencias adicionales; es un sitio web estático puro.
